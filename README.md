# Nuxt.js Frontend Application
このアプリケーションは、Nuxt.jsを使ったシンプルなToDoリストアプリです。

## 起動方法
### 前提条件
動作確認した環境は下記になります。
* Node.js v18.17.1
* npm 9.6.7
* Ubuntu

### インストール手順
1. リポジトリのクローン
```
git clone https://github.com/YuheiMuto/sample-todo-backend.git
```

2. アプリケーションのディレクトリに移動
```
cd sample-todo-backend
```

3. 依存関係のインストール
```
npm install
```

### アプリケーションの起動
1. 開発サーバーを起動
```
npm run start
```
